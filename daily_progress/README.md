# 🌅 Daily Progress System

매일 작은 진전을 이루고, 기록하고, 공유하는 시스템

---

## 사용법

### 아침: 하루 시작
```bash
cd ~/github/UNSEAM
claude "오늘 시작"   # 또는 "/daily start"
```

### 저녁: 하루 마무리
```bash
claude "오늘 마무리"  # 또는 "/daily end"
```

### 결과물
- `daily_progress/logs/YYYY-MM-DD.md` - 일일 기록
- 팀즈에 복붙할 수 있는 메시지 자동 생성

---

## 핵심 원칙

> **"완벽하지 않아도 괜찮아. 오늘 한 줄의 코드, 한 문장의 정리도 진전이야."**

- 🎯 목표는 **작게** 설정
- ✅ 완료는 **확실하게** 기록
- 💪 시작은 **격려와 함께**
