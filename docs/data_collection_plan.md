## 2. 실험 절차

### 2.1 전체 흐름

```
참가자 도착 (T-30분)
    ↓
동의서 설명 및 서명 (10분)
    ↓
설문지 작성 (20분)
  - STAI-T (특성 불안)
  - STAI-S-pre (상태 불안-사전)
  - IUS (불확실성 불내성)
  - BDI-II (우울)
  - Demographic info
    ↓
MRI 안전 체크 및 준비 (10분)
    ↓
MRI 스캔 (60분)
  - Localizer (2분)
  - T1-weighted (6분)
  - Resting-state fMRI (10-20분)
  - Field map (optional, 2분)
    ↓
스캔 후 설문 (5분)
  - STAI-S-post (상태 불안-사후)
  - Scan experience questions
    ↓
사례금 지급 및 종료 (5분)

총 소요 시간: ~1.5시간
```

### 2.2 상세 절차

#### **Step 1: 도착 및 오리엔테이션** (T-30 ~ T-20분)

**장소**: 대기실

**진행**:
1. 참가자 신원 확인
2. 연구 개요 설명:
   - 연구 목적
   - 절차
   - 예상 소요 시간
   - 위험성 및 이득
   - 중도 포기 가능
3. 질문 응답

#### **Step 2: 동의서** (T-20 ~ T-10분)

**문서**:
- 연구 참여 동의서
- 개인정보 수집 및 이용 동의서
- MRI 안전 동의서

**설명 사항**:
- 데이터 사용 범위
- 익명화 처리
- 보관 기간
- 중도 철회 권리

#### **Step 3: 설문지 작성** (T-10 ~ T+10분)

**순서**:
1. STAI-T (특성 불안) - 5분
2. STAI-S (상태 불안-사전) - 5분
3. IUS (불확실성 불내성) - 5분
4. BDI-II (우울) - 5분
5. Demographic questionnaire - 2분
   - 나이, 성별, 교육 수준
   - 수면 시간 (전날)
   - 카페인 섭취 (당일)
   - 스트레스 수준 (최근 1주)

**방법**: 태블릿 또는 종이

#### **Step 4: MRI 준비** (T+10 ~ T+20분)

**체크리스트**:
- [ ] 금속 물질 최종 확인
- [ ] 안경/시계/지갑/핸드폰 보관
- [ ] 화장실 이용
- [ ] MRI 룸 입장 안내
- [ ] 헤드폰/버튼박스 착용
- [ ] Emergency bulb 설명
- [ ] 편안한 자세 조정

**Instruction (한국어)**:
```
"이제 뇌 촬영을 시작하겠습니다.
촬영 중에는 큰 소음이 발생하지만 정상이니 놀라지 마세요.
최대한 움직이지 말고 편안히 계세요.

첫 번째는 구조 촬영입니다. 약 6분 소요됩니다.
두 번째는 안정 상태 촬영입니다. 약 10-20분 소요됩니다.
특별히 무엇을 생각하려 하지 마시고, 자연스럽게 두세요.

[Eyes open 조건]
화면에 십자가가 보일 겁니다. 십자가를 보세요.
깜박이는 것은 괜찮지만, 눈을 감지는 마세요.

[Eyes closed 조건]
눈을 편안히 감으세요.

불편하거나 중단하고 싶으면 왼손의 비상벨을 눌러주세요.
준비되셨나요?"
```

#### **Step 5: MRI 스캔** (T+20 ~ T+80분)

**Sequence 순서**:
1. **Localizer** (2분)
   - 목적: 스캔 위치 확인

2. **T1-weighted structural** (6분)
   - 목적: 해부학적 영상

3. **Resting-state fMRI** (10-20분)
   - 목적: 주요 데이터
   - Eyes open with fixation cross 권장

4. **Field map** (2분, optional)
   - 목적: Distortion correction

**스캔 중 모니터링**:
- 실시간 motion tracking
- Participant comfort check (중간에 1회)

#### **Step 6: 스캔 후** (T+80 ~ T+90분)

**설문**:
1. STAI-S (상태 불안-사후) - 3분
2. Scan experience - 2분
   - 스캔 중 졸음 정도 (1-5)
   - 스캔 중 불안 정도 (1-5)
   - 특정 생각/걱정 여부 (자유 응답)
   - 스캔 중 불편함 (자유 응답)

**사례금**:
- 현금 5만원 지급
- 영수증 서명

**Debriefing**:
- 연구 결과 공유 여부 확인
- 추가 연구 참여 의향 확인

**장비**: [구체적 스캐너 모델 입력]

- 예: Philips Ingenia 3T

**코일**: 32-channel or 64-channel head coil

### 3.2 Sequence Parameters

#### **T1-weighted MPRAGE**
```
TR: 2000 ms
TE: 2.5 ms
Flip angle: 9°
FOV: 256 × 256 mm
Matrix: 256 × 256
Slice thickness: 1 mm
Slices: 176 (sagittal)
Voxel size: 1 × 1 × 1 mm³
Scan time: ~6 min
```

#### **Resting-state EPI**

**Option A: Standard TR**
```
TR: 2000 ms
TE: 30 ms
Flip angle: 90°
FOV: 240 × 240 mm
Matrix: 80 × 80
Slice thickness: 3 mm
Gap: 0 mm
Slices: 40 (interleaved)
Voxel size: 3 × 3 × 3 mm³
Volumes: 300 (10 min) or 600 (20 min)
```

**Option B: Multiband (권장)**
```
TR: 720 ms
TE: 33 ms
Flip angle: 52°
Multiband factor: 8
FOV: 208 × 208 mm
Matrix: 104 × 104
Slice thickness: 2 mm
Slices: 72
Voxel size: 2 × 2 × 2 mm³
Volumes: 833 (10 min) or 1666 (20 min)
```

#### **Field Map (Optional)**
```
TR: 400 ms
TE1: 4.92 ms
TE2: 7.38 ms
Flip angle: 60°
FOV: 240 × 240 mm
Matrix: 80 × 80
Voxel size: 3 × 3 × 3 mm³
Scan time: ~2 min
```

### 3.3 스캔 순서 및 타이밍

| Sequence | Duration | Purpose | Notes |
|----------|----------|---------|-------|
| Localizer | 30 sec | Positioning | Standard |
| T1 MPRAGE | 6 min | Anatomy | High resolution |
| Resting-state | 10-20 min | Main data | Eyes open/closed |
| Field map | 2 min | Correction | Optional |
| **Total** | **18-28 min** | | |

### 3.4 참가자 Instruction

#### **Eyes Open (권장)**

**장점**:
- 각성 상태 유지 (졸음 방지)
- Standardized across participants

**화면 제시**:
- Black background
- White fixation cross (중앙)
- Cross size: 0.5° visual angle

**Instruction**:
"십자가를 보세요. 깜박이는 것은 괜찮지만 눈을 감지 마세요."

#### **Eyes Closed (대안)**

**장점**:
- 더 자연스러운 resting state
- Visual input 제거

**단점**:
- 졸음 위험 ↑
- Participant compliance 확인 어려움

**Instruction**:
"눈을 편안히 감으세요. 잠들지 않도록 주의하세요."

## 5. Quality Control

### 5.1 스캔 중 QC

**실시간 모니터링**:
- [ ] Signal intensity (SNR)
- [ ] Motion parameters (실시간 그래프)
- [ ] 참가자 compliance (카메라)

**중단 기준**:
- Excessive motion (FD > 3mm for >10% volumes)
- Signal dropout
- 참가자 요청

**재스캔 결정**:
- 즉시 재스캔 (시간 허용 시)
- 다음 방문 예약

### 5.2 스캔 후 QC

**자동 QC (스크립트)**:
```bash
# MRIQC 실행
mriqc data/processed/ data/derivatives/mriqc/ \
    participant --participant-label 001
```

**체크 항목**:
1. **Anatomical (T1)**:
   - [ ] SNR > 10
   - [ ] Contrast-to-noise ratio (CNR)
   - [ ] No severe artifacts
   - [ ] Full brain coverage

2. **Functional (EPI)**:
   - [ ] tSNR > 50
   - [ ] Mean FD < 0.3 mm
   - [ ] Max FD < 5 mm
   - [ ] % volumes with FD > 0.5 mm < 20%
   - [ ] DVARS
   - [ ] Temporal variability

**QC 리포트**:
- MRIQC HTML report 생성
- 주간 QC 미팅에서 리뷰

### 5.3 제외 기준

**Exclude if**:
1. Mean FD > 0.5 mm
2. >20% volumes with FD > 0.5 mm
3. Signal dropout in >10% brain
4. Severe artifacts (motion, susceptibility)
5. Participant reported excessive sleepiness (>4/5)

**재모집**:
- Excluded participants → 재모집
- Target N 유지

## 7. 문제 해결 (Troubleshooting)

### 문제 1: 참가자 No-show

**예방**:
- 스캔 1일 전 리마인더
- 취소 시 최소 24시간 전 연락 요청

**대응**:
- 대기자 리스트에서 빠른 대체
- No-show 참가자는 재참여 불가

### 문제 2: 과도한 Motion

**예방**:
- 헤드 패딩 충분히
- 편안한 자세 조정
- Instruction 명확히

**대응**:
- 즉시 재스캔 (가능 시)
- 불가 시 다음 방문 예약 (사례금 50% 지급)

### 문제 3: 참가자 불편/중단

**대응**:
- 즉시 스캔 중단
- 건강 상태 확인
- 사례금 50% 지급
- 재참여 의향 확인

### 문제 4: 기술적 문제 (스캐너)

**대응**:
- 스캐너 담당자에게 즉시 연락
- 참가자에게 상황 설명
- 다음 방문 재예약 (전액 사례금 지급)
