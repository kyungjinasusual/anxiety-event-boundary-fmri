## 데이터 모달리티 (Data Modality)

### 뇌영상
- **fMRI**: 과제 기반 (영화 시청) + 휴지기 상태
- **스캐너**: 3T Siemens Magnetom TIM Trio
- **시퀀스**: Multiband gradient-echo EPI
- **파라미터**:
  - TR = 1.3초
  - TE = 30ms
  - 복셀 크기: 2.5 × 2.5 × 2.5 mm³
  - 54 슬라이스
  - Multiband 가속 인자 = 3
- **구조영상**: 고해상도 T1 가중 해부학적 스캔

### 생리신호 기록
- **심박**: PPG (광용적맥파)
- **호흡**: 호흡 벨트
- **피부 전기 활동 (EDA)**: 피부 전도도
- **샘플링 레이트**: 1000 Hz

### 행동 주석
- **연속 평가**: 50개 감정 관련 항목
- **시청 후 설문**: 몰입도, 즐거움, 흥미
- **특질 설문**: DASS, BIS/BAS, ERQ, Big Five

## 자극 (Stimuli)

### 영화 자극
- **영화 수**: 14편의 단편 영화
- **총 길이**: 약 2.5시간 (150분)
- **평균 영화 길이**: 11분 26초
- **영화 예시**:
  - "Sintel" (Blender Foundation)
  - "Tears of Steel" (Blender Foundation)
  - "After the Rain"
  - 기타 감정을 유발하는 단편 영화들

### 주석된 감정 차원 (50개 항목)
**감정의 구성요소 과정 모델 (Component Process Model, CPM)** 기반:

1. **평가 (Appraisal)** - 인지적 평가
2. **동기 (Motivation)** - 행동 경향성
3. **운동 표현 (Motor expression)** - 얼굴, 음성, 몸짓
4. **생리적 반응 (Physiological response)** - 자율신경계 변화
5. **느낌 (Feeling)** - 주관적 경험

추가로 **개별 감정**: 행복, 슬픔, 두려움, 분노, 혐오 등

### 주석 신뢰도
- **평균 평가자 간 일치도**: 0.38 (연속 감정 평가 기준 중간 수준)

## 과제 (Task)

### 영화 시청 프로토콜
1. **스캔 전**: 참여자가 스캐너 환경에 적응
2. **영화 시청**: 스캐너 내에서 14편의 단편 영화 수동 시청
3. **휴지기 상태**: 영화 시청 전후
4. **시청 후**: 설문지 작성 (몰입도, 즐거움, 흥미)

### 주석 과제 (별도 코호트)
- **방법**: 맞춤 소프트웨어를 사용한 온라인 주석
- **참여자**: 44명 평가자 (원격 주석)
- **평가 항목**: 영화 재생 중 50개 감정 요소 연속 평가
- **인터페이스**: 실시간 업데이트되는 슬라이딩 척도

## 참여자 (Participants)

### fMRI 연구 (뇌영상)
- **N = 30명** (여성 18명, 남성 12명)
- **연령**: 18-35세 (평균 명시되지 않음)
- **건강한 대조군**: 예 - 정신/신경학적 질환 없음
- **포함 기준**:
  - 높은 영어 구어 이해력
  - 정신 또는 신경학적 질환 병력 없음
  - 약물 사용 없음
  - 현재 신경약리학적 약물 복용 없음
- **동일 참여자**: 예 - 행동 및 뇌 데이터가 동일 개인으로부터 수집됨

### 주석 연구 (행동)
- **N = 44명** (여성 23명, 남성 21명)
- **연령**: 20-39세 (평균 = 25.31)
- **포함 기준**: fMRI 연구와 동일
- **방법**: 원격 온라인 주석 (개인 컴퓨터 사용)

## 길이 (Length)

### 스캔 세션
- **영화 시청 런**: 약 150분 (14편 × 평균 11분)
- **휴지기 상태**: 명시되지 않음 (표준 5-10분)
- **해부학적 스캔**: 약 5-10분
- **총 스캔 시간**: 약 2.5-3시간

### 참여자당 데이터
- **영화 fMRI**: 약 6,900 볼륨 (150분 ÷ 1.3초 TR)
- **휴지기 fMRI**: 약 400-600 볼륨 (10분 가정)
- **생리신호**: 1000 Hz × 150-180분 = 900만~1000만 샘플

## 행동 데이터와 뇌 데이터 대응

**대응**: ✅ **완전 일치**

- **동일한 30명 참여자**가 제공:
  1. fMRI 데이터 (영화 시청 + 휴지기 상태)
  2. 생리신호 데이터 (심박, 호흡, EDA)
  3. 시청 후 행동 평가 (몰입도, 즐거움, 흥미)
  4. 특질 설문 응답 (DASS, BIS/BAS, ERQ, Big Five)

- **다른 44명 참여자**가 제공:
  - 연속 감정 주석 (오프라인 평가 과제)
  - 영화에 대한 "ground truth" 감정 라벨 생성에 사용

**데이터 구조**:
- 개인 내: fMRI ↔ 생리신호 ↔ 시청 후 평가 ↔ 특질
- 개인 간: fMRI 참여자 vs. 주석 참여자

## 행동 결과 변수 및 메타데이터

### 시청 후 평가 (30명 fMRI 참여자)
1. **몰입도 (Absorption)**: 영화에 얼마나 몰입했는지
2. **즐거움 (Enjoyment)**: 영화를 얼마나 좋아했는지
3. **흥미 (Interest)**: 영화가 얼마나 흥미로웠는지

### 연속 감정 주석 (44명 주석 참여자)
**50개 감정 항목** 연속 평가 (슬라이딩 척도, 실시간):

**개별 감정**:
- 행복, 슬픔, 두려움, 분노, 혐오, 놀람, 경멸

**평가 요소**:
- 새로움, 유쾌함, 목표 관련성, 대처 능력, 규범 적합성

**동기 요소**:
- 접근, 회피, 주의, 행동 준비성

**운동 표현**:
- 얼굴 표정, 음성 표현, 몸짓 표현

**생리적 반응**:
- 각성, 긴장, 에너지

**느낌 요소**:
- 정서가, 강도, 감정 특성

### 특질 설문지 (30명 fMRI 참여자)

| 설문지 | 측정 변수 | 분석 사용 |
|--------|----------|----------|
| **DASS-21** | 우울, 불안, 스트레스 | 부정적 정서의 개인차 |
| **BIS/BAS** | 행동 억제/활성화 | 동기 체계 |
| **ERQ** | 감정 조절 (재평가, 억압) | 감정 조절 전략 |
| **Big Five** | 개방성, 성실성, 외향성, 친화성, 신경증 | 성격 특질 |

### 생리신호 메타데이터
- **심박수**: 분당 박동수 (PPG에서 도출)
- **호흡수**: 분당 호흡수
- **심박 변이도 (HRV)**: RMSSD, SDNN
- **피부 전기 활동**: 피부 전도도 수준, 위상 반응

### fMRI 메타데이터 (BIDS 형식)
- Framewise displacement (움직임)
- DVARS (신호 변화 지표)
- 런당 평균 FD
- 이상치 볼륨 플래그

## 이론적 틀 (Theoretical Framework)

### 감정의 구성요소 과정 모델 (Component Process Model, CPM)

**참고문헌**: Scherer, K. R. (2009). The dynamic architecture of emotion: Evidence for the component process model. *Cognition and Emotion*, 23(7), 1307-1351.

**핵심 원리**:
1. **감정은 창발적 과정**이지 개별적 상태가 아님
2. **다섯 가지 동기화된 하위 체계**:
   - 평가 (인지적)
   - 동기 (행동 경향성)
   - 표현 (운동)
   - 생리 (신체 변화)
   - 느낌 (주관적 경험)
3. **역동적 전개**: 감정은 시간에 따라 연속적으로 변화
4. **요소 분리**: 하위 체계가 분리될 수 있음

**Emo-FilM과의 관련성**:
- 자연주의적 영화가 **복잡하고 역동적인 감정** 유발
- 50개 주석 항목이 CPM 구성요소에 매핑
- 다중모달 데이터(fMRI, 생리신호, 행동)가 다중 감정 체계 포착
- 연속 평가가 시간적 역동성 포착

### 자연주의적 패러다임 근거

**실험실 감정 유도 대신 영화를 사용하는 이유?**
1. **생태학적 타당도**: 영화가 실제 감정 맥락과 유사
2. **시간적 역동성**: 감정이 수 초가 아닌 수 분에 걸쳐 전개
3. **복잡한 평가**: 여러 감정 요소가 동시에 작용
4. **몰입**: 높은 몰입도가 요구 특성 감소
5. **재현 가능성**: 모든 참여자에게 동일한 자극

**해결된 과제들**:
- 개인 변산성 → 대규모 주석 샘플 (N=44)로 ground truth 확보
- 복잡성 → 구성요소 기반 측정 (CPM 틀)
- 움직임 아티팩트 → 짧은 클립, 흥미로운 콘텐츠로 움직임 최소화

## 데이터 분석 (Data Analysis)

### 전처리 파이프라인

**해부학적**:
- 뇌 추출
- 분할 (회백질, 백질, 뇌척수액)
- MNI152 공간으로 정규화

**기능적**:
- 움직임 보정 (MCFLIRT)
- 공간 평활화 (FWHM = 5mm)
- 고역 통과 시간 필터링 (100초 차단)
- 조직 클래스 회귀 (뇌척수액, 백질 잡음)
- 움직임 파라미터 회귀 (6 DOF + 미분)

**생리신호**:
- 심장 주기 감지 (R-피크)
- 호흡 주기 추출
- EDA 분해 (긴장성, 위상성 요소)
- fMRI TR로 다운샘플링 (1.3초)

**BIDS 준수**:
- 표준화된 디렉토리 구조
- JSON 메타데이터 파일
- 이벤트 파일 (영화 시작, 주석)

### 권장 분석 접근법

**1. 일변량 GLM**:
- 감정 회귀변수 (50개 연속 주석)
- HRF와 합성곱
- 개별 감정에 대한 대비 코딩
- 그룹 수준 혼합효과 모델

**2. 다변량 패턴 분석 (MVPA)**:
- 분산 패턴에서 감정 디코딩
- Searchlight 분석
- 영화 간 교차 검증
- 패턴 유사성 분석 (RSA)

**3. 동적 기능 연결성**:
- 슬라이딩 윈도우 (20-30초)
- 시간 해상 연결성 행렬
- 시간에 따른 그래프 지표
- 감정 주석과의 상관

**4. 생리적 결합**:
- 뇌-심장 결합
- 호흡 위상이 BOLD에 미치는 영향
- EDA-감정 상관
- 말초-중추 동기화

**5. 개인차**:
- 특질 조절 (DASS, BIS/BAS, ERQ, Big Five)
- 계층적 모델
- 개인 중심 분석

### 통계적 고려사항

**검정력 분석**:
- N=30은 중간-큰 효과 (d > 0.5)에 적절
- 개인 내 설계가 검정력 증가
- 여러 영화가 강건성 제공

**다중 비교**:
- FWE 보정 (클러스터 기반)
- FDR 보정 (복셀별)
- 비모수적 추론을 위한 순열 검정

**혼합효과 모델**:
- 고정효과: 감정, 영화 특징
- 무선효과: 참여자, 영화
- 중첩 구조 고려

## Event Boundary 연구 관련성

### 잠재적 활용

✅ **강점**:
- 서사 구조가 있는 자연주의적 자극
- 연속 감정 평가가 시간적 역동성 포착
- 고품질 fMRI (TR=1.3초, multiband)
- 각성 마커를 위한 생리신호 데이터
- DASS 불안 점수 이용 가능

⚠️ **한계**:
- **명시적 event boundary 주석 없음**
- 감정 평가 ≠ event boundary 라벨
- 영화 구조가 다양함 (표준화된 event 타이밍 없음)
- 불안 하위집단 분석에는 샘플이 작음 (N=30)
- 건강한 집단 (임상적 불안 장애 없음)

### 가능한 적응

**옵션 1: 감정 변화에서 boundary 라벨 도출**
- 감정 평가에 gradient 분석 사용
- 감정 변화 피크 = 후보 boundary
- 서사 분할과 대조하여 검증

**옵션 2: 서사 event 코딩**
- 장면 변화, 캐릭터 전환 수동 코딩
- 감정 역동성과 상관
- 불안이 boundary 탐지를 조절하는지 검증

**옵션 3: 검증 세트로 사용**
- 주요 분석: Spacetop (더 큰 N, 불안 측정)
- 검증: Emo-FilM (자연주의적 과제 맥락)

## 불안-Event Boundary 연구 요약 평가

| 기준 | 평가 | 비고 |
|-----|------|-----|
| **샘플 크기** | ⚠️ 보통 | N=30은 적절하나 불안 하위집단에는 이상적이지 않음 |
| **불안 측정** | ⚠️ 제한적 | DASS-21 (스크리닝), 진단 아님; 건강한 집단 |
| **Event 구조** | ⚠️ 불명확 | 영화에 서사는 있으나 명시적 boundary 라벨 없음 |
| **데이터 품질** | ✅ 우수 | 높은 시간 해상도 (TR=1.3초), multiband, BIDS |
| **다중모달** | ✅ 우수 | fMRI + 생리신호 + 행동 + 특질 |
| **이용 가능성** | ✅ 우수 | 공개 (OpenNeuro), CC0 라이선스 |
| **문서화** | ✅ 우수 | BIDS, 상세한 방법, 전처리 코드 |

**권장사항**:
- **부차적 검증 데이터셋** (주요 데이터셋 아님)
- 주요로 Spacetop (N=100-200, 더 강한 불안 측정) 사용
- Spacetop 결과가 강건하면 Emo-FilM으로 자연주의적 과제 검증
